# Keen Keeper

[![Netlify Status](https://api.netlify.com/api/v1/badges/your-netlify-badge-id/deploy-status)](https://assinment-7-keen-keeper-saad.netlify.app/)

## 📖 Description
**Keen Keeper** is a modern, responsive web application built to help users efficiently manage, track, and organize their data. Whether it's tracking tasks, keeping inventory, or managing schedules, this platform provides a seamless experience. 

**Live Demo:** [View Keen Keeper Here](https://assinment-7-keen-keeper-saad.netlify.app/)

## 🛠️ Technologies Used

**Build Tool:** Vite
**Frontend Framework:** React.js
**Routing:** React Router DOM
**Styling:** Tailwind CSS
**Icons:** Lucide-react
**Data Visualization:** Recharts
**Notifications:** React Hot Toast / React-Toastify (Toast notification er jonno)

## ✨ Key Features
1. **Interactive & Responsive UI:** A clean, user-friendly interface that works perfectly across desktops, tablets, and mobile devices.
2. **Dynamic Data Management:** Users can easily add, edit, or remove items with instantaneous updates reflecting on the screen.
3. **Optimized Performance:** Fast load times and smooth routing ensure a highly responsive and reliable user experience.
